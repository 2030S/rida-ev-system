#  EV Charging System – Morocco

A full-stack EV (Electric Vehicle) charging management system designed to connect charging stations, manage sessions, and enable smart charging operations across Morocco.

---

##  Project Overview

This project aims to build a scalable EV charging ecosystem that includes:

- Charging station management
- User session tracking
- Smart charging communication (OCPP-based backend)
- Mobile application for end users
- Central backend API for orchestration

The system is designed to be modular and extensible for future integration with payment systems and energy optimization.
